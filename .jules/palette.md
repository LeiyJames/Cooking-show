## 2025-04-02 - Accessible Grouped Steppers with Conditional Framer Motion
**Learning:** When applying disabled states to interactive Framer Motion components (like `motion.button`), the visual `disabled:opacity-50` isn't enough. The `whileHover` and `whileTap` properties still apply tactile feedback to the disabled element, making it feel interactable. Additionally, grouped interactive elements like steppers need a unifying `role="group"` and `aria-labelledby` linked to their visual label using `React.useId()`, while dynamic outputs need `aria-live="polite"`.
**Action:** When creating steppers or visually disabling Framer Motion elements, always conditionally apply motion props (e.g., `whileHover={isDisabled ? {} : { scale: 1.05 }}`) to remove physical interactivity, and use `role="group"` with proper IDs to link the stepper to its label.
